Features

Everything You Need to Manage Sales

Complete and professional interface with advanced pipeline management features

📋

Interactive Kanban Board

Visualize your pipeline in 5 customizable stages with drag-and-drop interface. Move leads between columns effortlessly with smooth animations.

📊

Real-Time Statistics

Dashboard with instantly updated metrics: total leads, pipeline value, closed value, conversion rate, and closed deals.

⚡

Complete Lead Management

Add and edit leads through complete modal forms. Double-click to edit, right-click for context menu.

🔍

Search & Filters

Filter cards by name, company, or email in real-time. Instant search that updates the Kanban automatically.

🎯

Priority System

HIGH/MEDIUM/LOW classification with color visual indicators. Quickly identify the most important deals.

💾

Data Persistence

Automatic saving to local JSON on every change. Data automatically restored on next launch.

Getting Started

How to Run the Project

Simple and quick installation in just a few commands

1

Clone the Repository

Download the source code from GitHub

git clone https://github.com/ChristopherDond/CRM-Pipeline-Flow.git
2

Navigate to Project

Enter the application directory

cd CRM-Pipeline-Flow/crm-pipeline
3

Run with Maven

Compile and run the application

mvn javafx:run

⚠️ Requirements: Java JDK 21+ and Apache Maven 3.9+

On first run, Maven will download JavaFX and Gson dependencies. This may take 1-2 minutes.
